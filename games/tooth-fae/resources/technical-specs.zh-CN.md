## 技术堆栈

### 编程

**主要语言**：Nim
- 现代系统编程语言
- 专注于效率和表达力
- 编译为 C 以提高性能

**图形框架**：SDL2
- 跨平台多媒体库
- 硬件加速 2D 渲染
- 输入处理和事件管理

### 美术与图形

**精灵创作**：Asesprite
- 专业像素艺术编辑器
- 动画时间线支持
- 精灵表导出功能
- 像素艺术游戏的行业标准工具

**美术风格**：像素艺术
- 手工制作的 2D 精灵
- 氛围角色和环境设计
- 深色、忧郁的配色方案
- 流畅的逐帧动画

### 音频制作

**音乐创作**：Ableton Live
- 专业数字音频工作站（DAW）
- 从头开始创作的原创配乐
- 氛围恐怖音乐设计
- 音效创作和混音

**附加音频工具**：Audacity
- 开源音频编辑
- 音效处理
- 音频清理和母带处理

## 平台支持

### HTML5（浏览器版本）

**支持的浏览器**：
- Google Chrome（推荐）
- Mozilla Firefox（推荐）
- Safari（有注意事项 - 见下文）
- Microsoft Edge
- 其他具有 HTML5 支持的现代浏览器

**浏览器要求**：
- HTML5 canvas 支持
- WebGL 支持（推荐）
- JavaScript 已启用
- 现代浏览器（过去 2 年内发布）

**Safari 特定注意事项**：
- 禁用低功耗模式以防止限制
- 性能可能与 Chrome/Firefox 相比有所不同
- 建议使用替代浏览器以获得最佳体验

### Windows 桌面版

**下载大小**：24 MB

**文件格式**：.zip 压缩包
- 解压到任何位置
- 无需安装
- 便携式可执行文件

**系统要求**（估计）：
- **操作系统**：Windows 7 或更高版本
- **处理器**：现代双核处理器
- **内存**：最少 2 GB RAM
- **图形**：任何具有基本 2D 加速的 GPU
- **存储**：100 MB 可用空间
- **附加**：键盘/鼠标或手柄

**浏览器版本**：
- **CPU**：轻到中度

**桌面版本**：
- **CPU**：轻

## 构建过程

### 编译流程

1. **源代码**：Nim 源代码
2. **编译**：Nim → C
3. **原生二进制**：C → 平台可执行文件
4. **Web 构建**：SDL2 → Web（HTML5/WebAssembly）

### 开发工具链

```
Nim 语言 → SDL2 框架 → 平台二进制
     ↓              ↓
 Asesprite      Ableton Live
 （图形）         （音频）
```
